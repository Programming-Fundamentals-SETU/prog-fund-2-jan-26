## Part 3: Linking IntelliJ IDEA to GitHub

### Step 1: Open IntelliJ IDEA

1. Open **IntelliJ IDEA**
2. Go to:
   - **File → Settings** (Windows)
   - **IntelliJ IDEA → Settings** (Mac)

### Step 2: Connect to GitHub

1. Select **Version Control → GitHub**
2. Click **Add Account**
3. Choose **Log in via GitHub**
4. A browser window will open
5. Log in to GitHub and authorise IntelliJ

Once complete, IntelliJ is linked to your GitHub account.
