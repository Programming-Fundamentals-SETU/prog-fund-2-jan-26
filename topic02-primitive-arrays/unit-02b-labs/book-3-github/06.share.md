
## Part 6: Sharing the Project on GitHub

1. In IntelliJ, go to **Git → GitHub → Share Project on GitHub**
2. Enter a repository name (e.g. `GitHubIntro`)
3. Leave the repository as **Private** (unless instructed otherwise)
4. Click **Share**

IntelliJ will:
- Create the repository on GitHub
- Commit your files
- Push the project to GitHub



## Part 7: Making a Change and Pushing It

### Step 1: Add Code
1. Open `Main.java`
2. Add the following code:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello GitHub!");
    }
}
```

### Step 2: Commit and Push

1. IntelliJ will highlight changed files
2. Click **Commit**
3. Enter a commit message:
   ```
   Added hello message
   ```
4. Click **Commit and Push**
5. Confirm the push
