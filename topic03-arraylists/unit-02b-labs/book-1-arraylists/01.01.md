# 2. Starting ShopV3.0

In this step, you will use the [ShopV2.2 Solution](./archives/ShopV2.2.zip) as a starting point for Shop V3.0.  In this version:

- you will remove the primitive array and use an ArrayList instead.


## Creating ShopV3.0

In IntelliJ, create a new project called `ShopV3.0`.  

Copy the src files (Driver, Store, Product) from ShopV2.2 and paste them in the src for ShopV3.0.

![ShopV3.0 folder structure](./img/01.png)


## Test the code base

Run your version of ShopV3.0 to both test it and also re-familiarise yourself with the project.
