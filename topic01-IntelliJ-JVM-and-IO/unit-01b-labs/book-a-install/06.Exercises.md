# Exercises

## Exercise 1:

Examine the Hello World project folder in Explore/Finder - and compare the contents with the view presented in Idea.

## Exercise 2:

Create another new project - like Hello World - but give the project a different name. Add different print statements.  Get used to having more than one project open at a time.

## Exercise 3:

Create another new project, called MyInformation. Add 5 print statments, each should have a piece of information about you.

e.g. 
~~~
My name is Siobhan
My dogs name is Buddy
My daughter wants a kitten (not happpening!)
I talk (a lot)
I don't like typing though
~~~

The following are copied from the Induction, no need to complete if you have already done them
