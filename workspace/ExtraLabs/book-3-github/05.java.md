## Part 4: Creating a Java Project in IntelliJ

1. From the IntelliJ welcome screen, click **New Project**
2. Select **Java**
3. Ensure a JDK is selected
4. Click **Next** / **Create**
5. Name your project (e.g. `GitHubIntro`)
6. Choose a suitable location on your computer
7. Click **Create**
