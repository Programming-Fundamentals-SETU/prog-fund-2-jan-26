# ShopV8.0 

In this lab, you will create version 8 of the Shop App.  In this version, we will add a new section onto the menu for Searching and Sorting products.

## Creating ShopV8.0 Project

Use either your V7.1 of the project or this downloadable version: [ShopV7.1.zip](archives/ShopV7.1.zip).

Open Windows Explorer / Mac Finder and locate where V7.0 is.  Copy the ShopV7.1 folder and paste it with the new name ShopV8.0.  

Open this new project in IntelliJ.  Once in IntelliJ, you will notice that the Project name is still ShopV7.1.  Right click on it and select **Refactor...Rename**.  Call the project ShopV8.0.

Now we are ready to start adding the new Menu section. 


## New Menu Options

Currently, the menu options are:

~~~
                ------------------------------------------------------------------
                |                            Shop Menu                           |
                ------------------------------------------------------------------
                | PRODUCT MENU                                                   | 
                |   1) Add a product                                             |
                |   2) List the Products                                         |
                |   3) Update a product                                          | 
                |   4) Delete a product                                          | 
                ------------------------------------------------------------------
                | PRODUCT DESCRIPTION MENU                                       | 
                |   5) Add a product description                                 |
                |   6) List product description(s)                               |
                |   7) Update a product description                              | 
                |   8) Delete a product description                              | 
                ------------------------------------------------------------------
                | REPORT MENU                                                    | 
                |   10) List the current products                                |
                |   11) Display average product unit cost                        |
                |   12) Display cheapest product                                 |
                |   13) List products that are more expensive than a given price |
                ------------------------------------------------------------------
                |   20) Save products to products.xml                            |  
                |   21) Load products from products.xml                          |  
                |   0)  Exit                                                     |  
                ------------------------------------------------------------------
                ==>> 
~~~

Add the following new menu options in mainMenu():

~~~
                ------------------------------------------------------------------            
                | SEARCH AND SORT MENU                                           | 
                |   14) Search products by name                                  |
                |   15) Sort products by name ascending                          |
                ------------------------------------------------------------------  
~~~

Add two empty methods:

~~~
    private void searchProductByName() {
    }

    private void sortProductByName() {
    }
~~~

And call them from the switch statement in runMenu():

~~~
                case 14 -> searchProductByName();
                case 15 -> sortProductByName();
~~~
