## NewsFeed

In your NewsFeed class, the current show() method is like this:

~~~
    public String show() {
        String str = "";

        for(Post post: posts) {
            str += posts.indexOf(post) + ": " + post.display() + "\n";
        }

        if (str.isEmpty()){
            return "No Posts";
        }
        else {
            return str;
        }
    }
~~~

Refactor it to use displayCondensed() instead:

~~~
    public String show() {
        String str = "";

        for(Post post: posts) {
            str += posts.indexOf(post) + ": " + post.displayCondensed() + "\n";
        }

        if (str.isEmpty()){
            return "No Posts";
        }
        else {
            return str;
        }
    }
~~~
