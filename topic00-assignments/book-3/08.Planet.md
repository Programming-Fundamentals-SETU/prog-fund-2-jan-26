## Abstract methods

`calculateGravity` - this method returns a (double) value for the gravity.  The algorithm for calculating this is: 
~~~
        // Algorithm - 
        //  (mass multipled by 6.67430e-11) divided by (half the diameter to the power of 2)
~~~
